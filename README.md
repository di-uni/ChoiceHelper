# ChoiceHelper
선택이 어려운 사람들을 위한 랜덤선택 서비스

<br/><br/>

## 기술스택
- MERN (MongoDB, Express, React js, Node js)

<br/><br/>

## How to run the project
```
$ git clone https://github.com/ksa37/ChoiceHelper.git
$ cd server
$ npm run start
$ (another terminal)
$ cd client
$ npm run start
```
Open http://localhost:3000 in browser to access this program.
